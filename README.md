## Projeto 2 — Automação para escritório de advocacia

### Contexto

O escritório precisava organizar dados de clientes recebidos por API e distribuir essas informações em diferentes planilhas de controle.

### Solução criada

Criei um fluxo no n8n que recebia os dados via API, realizava leitura/consulta de CNPJ, tratava as informações e salvava os dados dos clientes em 3 planilhas diferentes.

### Resultado

A automação reduziu trabalho manual, padronizou o registro dos dados e melhorou a organização das informações utilizadas pela operação.

### Competências aplicadas

- n8n
- API
- Webhook
- Consulta de CNPJ
- Tratamento de dados
- Integração com planilhas
- Organização operacional
- Automação de processos internos

## Projeto 3 — Agente de atendimento e vendas para Positivo Brasil

### Contexto

A operação precisava de um fluxo automatizado para atendimento e apoio a vendas, especialmente em momentos de campanha com alto volume de interações.

### Solução criada

Criei um fluxo/agente automatizado que recebia as interações, processava o contexto, conduzia o atendimento, apoiava o processo de vendas e direcionava o usuário conforme regras definidas.

### Resultado

Durante períodos de campanha, o fluxo chegou a ultrapassar 40.000 execuções, apoiando a operação em escala.

### Competências aplicadas

- n8n
- Agente automatizado
- Atendimento digital
- WhatsApp
- Vendas
- Campanhas
- Regras de fluxo
- Monitoramento de execuções
- Operação em alto volume
